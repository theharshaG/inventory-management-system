# inventory-management-system

Inventory Management System (Python Project)

## Overview

This project is a simple command-line Inventory Management System implemented in Python. It allows users to add products, view available products, update stock, and delete products. All product data is stored in a text file to maintain persistence between program executions.

## Features

Add new products
View all products
Update product stock
Delete products
Data persistence using file handling

## Technologies Used

Python
Object-Oriented Programming (OOP)
File Handling

## How to Run

install python (VS Code)
Run the program: inventory management system.py

## How It Works

Each product has a name and quantity
Data is stored in a text file (inventory.txt) in the format:
name,quantity

When the program starts, it reads the file and loads products into objects
Users can add, update, view, or delete products
When the program exits, all data is saved back to the file
